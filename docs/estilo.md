## 📏 Normas de Estilo

Para que el manual mantenga una coherencia, intenta seguir estas pautas:

* **Tono amigable:** Escribe de forma clara y cercana, evitando tecnicismos innecesarios.
* **Formato Markdown:** Usa negritas para destacar elementos de la interfaz (ej: "Haz clic en **Guardar**").
* **Imágenes:** Si añades imágenes, guárdalas en la carpeta `docs/assets/` y asegúrate de que no pesen demasiado.
* **Cuadros de aviso:** Usa los avisos de MkDocs para destacar info importante:

    ```markdown
    !!! info "Información"
        Usa esto para datos relevantes.

    !!! warning "Cuidado"
        Usa esto para evitar que el usuario cometa un error grave.
    ```
