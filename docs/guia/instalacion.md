# 📥 Instalación y Configuración

En esta sección te guiaremos paso a paso para que tengas **[Nombre de tu App]** lista para funcionar en menos de 5 minutos.

---

## 📋 Requisitos Previos

Antes de empezar, asegúrate de cumplir con lo siguiente:

* **Sistema Operativo**: Windows 10+, macOS 12+ o Linux.
* **Conexión a Internet**: Necesaria para la activación inicial.
* **Espacio en disco**: Al menos 200MB libres.

!!! info "Nota para Desarrolladores"
    Si planeas usar nuestra API, asegúrate de tener instalada la versión de Node.js 18 o superior.

---

## 🚀 Pasos para la Instalación

### Paso 1: Descarga del paquete
Accede a nuestro [sitio oficial](https://tuapp.com/descargas) y baja la versión correspondiente a tu sistema.

### Paso 2: Ejecución del instalador
Una vez descargado, sigue las instrucciones según tu sistema:

=== "Windows"
    Ejecuta el archivo `.exe` y concede los permisos de administrador cuando se te solicite.

=== "macOS"
    Arrastra el icono de la aplicación a tu carpeta de **Applications**.

=== "Linux"
    Dale permisos de ejecución al archivo `.AppImage`:
    ```bash
    chmod +x MiApp.AppImage
    ./MiApp.AppImage
    ```

---

## ✅ Verificación
Para comprobar que todo se ha instalado correctamente, abre la aplicación y verifica que aparezca el mensaje:  
`"Estado: Listo para trabajar"` en la esquina inferior izquierda.

!!! warning "⚠️ Importante"
    Si el antivirus bloquea la ejecución, añade una excepción para el ejecutable de la aplicación. Es un falso positivo común en versiones beta.
